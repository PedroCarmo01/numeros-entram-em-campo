# Os números entram em campo?

Projeto desenvolvido em Power BI para analisar se o desempenho das seleções na fase de grupos da Copa do Mundo de 2022 influencia o sucesso nas fases eliminatórias.

## Objetivo

Investigar a relação entre os indicadores de desempenho das seleções na fase de grupos e a fase alcançada por elas ao longo do torneio.

## Ferramentas utilizadas

- Power BI
- Power Query
- DAX
- ETL (Extração, Transformação e Carga de Dados)
- Desenvolvimento de KPIs

## Metodologia

Foi criado um índice de desempenho baseado em:

- Pontos conquistados
- Saldo de gols
- Gols marcados

A partir desse índice, as seleções foram classificadas de acordo com seu desempenho na fase de grupos e comparadas com a fase alcançada na competição.

## Principais resultados

- 87,5% das seleções com melhor desempenho avançaram para o mata-mata.
- As duas seleções finalistas estavam entre as cinco equipes com melhor desempenho na fase de grupos.
- Os indicadores analisados sugerem uma forte relação entre o desempenho inicial e o avanço na competição.
